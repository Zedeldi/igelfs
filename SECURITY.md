# Security Policy

**_This project is not affiliated with IGEL Technology._**

## Reporting a Vulnerability

Please report any vulnerabilities discovered within IGEL OS to IGEL directly,
according to their [online instructions](https://kb.igel.com/security-safety/current/reporting-vulnerabilities-in-igel-products).

If the vulnerability concerns this project specifically, please [raise an issue on GitHub](https://github.com/Zedeldi/igelfs/issues/new).

## Security Notices

For a list of already disclosed vulnerabilities, please see the
[IGEL Product Security Information](https://kb.igel.com/security-safety/current/igel-product-security-information) page.
